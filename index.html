Importación a Odoo ERP

La base de datos en formato CSV que hemos construido está lista para importarse en el módulo de CRM o Contactos de Odoo. Aquí tienes la guía paso a paso y la correspondencia de campos recomendada:

1

Descarga el archivo CSV

Utiliza el archivo base_empresas_procesos_chile.csv generado en la carpeta.

2

Ir a Odoo CRM

Entra a Odoo > CRM o Contactos, haz clic en Favoritos > Importar registros.

3

Subir y mapear campos

Sube el archivo y mapea las columnas. Para evitar crear campos personalizados en Odoo, te sugerimos mapear el contenido técnico al campo Descripción (Notas internas), o crear campos personalizados tipo texto.

Columna CSV Campo Sugerido en Odoo Tipo de Campo
Empresa Compañía / Nombre (name) Estándar
RUT RUT (vat) Estándar
Pagina_Web Sitio Web (website) Estándar
Email Correo Electrónico (email) Estándar
Sector Categoría / Etiquetas (x_sector) Etiqueta o Texto
Proceso_Critico, Oportunidad_... Descripción / Notas (description) Texto Multilínea

Script Automatizado de Sincronización

Si deseas automatizar la subida de estos prospectos directamente a Odoo mediante su API XML-RPC oficial sin pasar por la interfaz web, ejecuta este script en Python. Creará un lead en el CRM para cada empresa:

import csv
import xmlrpc.client

# ---- CONFIGURACIÓN ODOO ----
ODOO_URL = "https://tu-base-odoo.odoo.com"
ODOO_DB = "nombre_base_datos"
USERNAME = "tu_usuario@email.com"
API_KEY = "tu_token_o_password"

# 1. Conexión y Autenticación con Odoo API
print("Conectando a Odoo...")
common = xmlrpc.client.ServerProxy(f"{ODOO_URL}/xmlrpc/2/common")
uid = common.authenticate(ODOO_DB, USERNAME, API_KEY, {})
if not uid:
    print("Error de autenticación. Verifica tus credenciales.")
    exit()
print(f"Autenticado con éxito. UID: {uid}")

models = xmlrpc.client.ServerProxy(f"{ODOO_URL}/xmlrpc/2/object")

# 2. Leer base de datos CSV de empresas
with open('base_empresas_procesos_chile.csv', mode='r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        # Generar descripción rica para el lead
        desc = (
            f"RUT: {row['RUT']}\n"
            f"Sector: {row['Sector']}\n"
            f"Región/Comuna: {row['Region']}, {row['Comuna']}\n"
            f"Proceso Crítico: {row['Proceso_Critico']}\n\n"
            f"OPORTUNIDADES DE VENTA:\n"
            f"- CONTROL: {row['Oportunidad_Control']}\n"
            f"- INSTRUMENTACIÓN: {row['Oportunidad_Instrumentacion']}\n"
            f"- ELÉCTRICA: {row['Oportunidad_Electrica']}\n\n"
            f"Contacto: {row['Email']} | LinkedIn: {row['LinkedIn']}\n"
            f"Sucursales físicas: {row['Sucursales']}"
        )
        
        # Crear Lead en CRM de Odoo
        lead_data = {
            'name': f"Prospección SUPCON: {row['Empresa']}",
            'partner_name': row['Empresa'],
            'email_from': row['Email'],
            'website': row['Pagina_Web'],
            'description': desc,
            'type': 'lead'
        }
        
        lead_id = models.execute_kw(
            ODOO_DB, uid, API_KEY, 
            'crm.lead', 'create', [lead_data]
        )
        print(f"Lead creado exitosamente para {row['Empresa']} (ID: {lead_id})")

print("Sincronización finalizada exitosamente.")
