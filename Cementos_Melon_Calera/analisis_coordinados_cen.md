# Registro y Análisis de Empresas Coordinadas (CEN Chile)

Este informe presenta el catastro completo y estructurado de las empresas registradas ante el **Coordinador Eléctrico Nacional (CEN)**. Se analizan de forma exhaustiva sus roles operativos y se identifican las oportunidades comerciales específicas para la comercialización de sistemas DCS, SIS, PPC, BESS EMS y SAS/RTU de SUPCON.

---

## 1. Resumen de Coordinados Únicos Registrados

La base de datos oficial arroja un total de **772 empresas coordinadas activas** interconectadas al Sistema Eléctrico Nacional (SEN). A continuación, se muestra su distribución y el potencial de venta para SUPCON:

| Categoría CEN | Cantidad de Empresas | Enfoque de Venta / Propuesta de Valor SUPCON |
| :--- | :---: | :--- |
| **Generación** | 617 | Suministro de controladores de planta de energía (PPC), gestión de almacenamiento de baterías (BESS EMS) y DCS para hidroeléctricas/térmicas. |
| **Transmisión** | 36 | Automatización de subestaciones (SAS), pasarelas de comunicación bajo protocolo IEC 61850 y Unidades Terminales Remotas (RTU). |
| **Distribución** | 6 | Sistemas SCADA de distribución, control y teleoperación de reconectadores y celdas de media tensión. |
| **Cliente Libre** | 113 | DCS de procesos y sistemas instrumentados de seguridad (SIS/ESD SIL3) en grandes industrias, mineras, papeleras y químicas. |
| **TOTAL** | **772** | **Propuesta de valor unificada e interconexión OT-IT.** |
